# 系統說明

首先先

```
git clone https://github.com/Beginneraboutlife116/the-f2e-2022-week2-dot-dot-sign.git
```

```
cd the-f2e-2022-week2-dot-dot-sign
```

```
npm install
```

```
npm run dev
```

可以在網址使用 `http://localhost:3000`
